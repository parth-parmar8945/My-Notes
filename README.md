This is a **Notes Management Web Application** where users can create, edit, and delete notes easily. The left sidebar provides quick access to **Add Note, Calendar, Archive, and Trash** options.
The top section displays **Recent Folders** along with their creation dates.
Users can create **new folders** to organize notes efficiently.
The **My Notes** section shows notes in a clean card-based layout.
Each note includes **edit and delete icons** for quick actions.
Notes are stored using **localStorage**, so data remains saved even after refreshing the page.
